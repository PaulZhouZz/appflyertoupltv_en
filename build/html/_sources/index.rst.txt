Guides of pushing the raw data on AppsFlyer to UPLTV
====================================================


1.Login AppsFlyer；

2.Enter "Integration > API Access"；

3.Find “Push API” part and set UPLTV as the Endpoint；

a. Push API version: **1.0**

b. HTTP method: **POST**

c. Endpoint: **http://34.249.72.4/upltv/**

d. Event messages: tick the **Install** part
